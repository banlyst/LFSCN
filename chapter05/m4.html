5.25. M4-1.4.16

M4 程序包包含一个宏处理器。

预计编制时间: 0.2 SBU
需求磁盘空间: 11.6 MB

5.25.1. 安装 M4

准备编译 M4： 

./configure --prefix=/tools

编译程序包： 

make

编译现在完成了。前面讨论过，没有必要在这一章中给临时工具运行测试套件。如果无论如何要运行 M4 的测试套件，执行以下指令： 

make check

安装程序包： 

make install

有关此程序包的细节位于第 6.26.2 节 “M4 的内容”。
