5.22. Gettext-0.18.1.1

The Gettext package contains utilities for internationalization and localization. These allow programs to be compiled with NLS (Native Language Support), enabling them to output messages in the user's native language.

预计编制时间: 0.6 SBU
需求磁盘空间: 101 MB

5.22.1. 安装 Gettext

对于我们的临时工具，我们只需要编制安装 Gettext 的一个二进制。

准备编译 Gettext： 

cd gettext-tools
EMACS="no" ./configure --prefix=/tools --disable-shared

配置选项的含义：

EMACS="no"

此项防止配置脚本获知安装 Emacs Lisp 文件的位置，因为测试已知在某些宿主上会挂起。

--disable-shared

我们现在不需要安装任何的 Gettext 共享库，也就不必编制。

编译程序包： 

make -C gnulib-lib
make -C src msgfmt

因为只编译了一个二进制，就不可能在不编译 Gettext 程序包中额外支持库的前提下运行测试套件。因此在此处不推荐尝试运行测试套件。

安装 msgfmt 二进制： 

cp -v src/msgfmt /tools/bin

有关此程序包的细节位于第 6.44.2 节 “Contents of Gettext”。
