5.17. Coreutils-8.19

The Coreutils package contains utilities for showing and setting the basic system characteristics.

预计编制时间: 0.7 SBU
需求磁盘空间: 126 MB

5.17.1. 安装 Coreutils

准备编译 Coreutils： 

./configure --prefix=/tools --enable-install-program=hostname

配置脚本的含义：

--enable-install-program=hostname

此项启用编制和安装 hostname 二进制文件 – 它默认被禁用，但被 Perl 的测试套件需求。

编译程序包： 

make

编译现在完成了。前面讨论过，没有必要在这一章中给临时工具运行测试套件。如果无论如何要运行 Coreutils 的测试套件，执行以下指令： 

make RUN_EXPENSIVE_TESTS=yes check

RUN_EXPENSIVE_TESTS=yes 参数告诉测试套件运行几个额外的测试，它们在某些平台上相对昂贵 (CPU 运算能力和内存使用)，但是在 Linux 上通常不是问题。

安装程序包： 

make install

以上命令拒绝安装 su，因为此程序无法作为非特权用户安装为 setuid root。通过手动安装，我们就可以用它来在最终系统中作为非特权用户运行测试。如此安装之： 

cp -v src/su /tools/bin

有关此程序包的细节位于第 6.26.2 节 “Contents of Coreutils”。
