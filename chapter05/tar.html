5.31. Tar-1.26

Tar 程序包包含一个归档程序。

预计编制时间: 0.3 SBU
需求磁盘空间: 20.9 MB

5.31.1. 安装 Tar

准备编译 Tar： 

./configure --prefix=/tools

编译程序包： 

make

编译现在完成了。前面讨论过，没有必要在这一章中给临时工具运行测试套件。如果无论如何要运行 Tar 的测试套件，执行以下指令： 

make check

安装程序包： 

make install

有关此程序包的细节位于第 6.58.2 节 “Tar 的内容”。
